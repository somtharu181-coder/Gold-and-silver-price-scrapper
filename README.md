# Gold & Silver Price Scraper

A lightweight Python script that automatically fetches real-time gold and silver rates via API-based extraction, then uses Pandas to clean, structure, and log the data for historical trend analysis.

##  Overview

This is a single-file script — no project structure, virtual environment, or configuration files required. Just run the script directly to fetch and log the latest gold and silver prices.

##  Features

- Fetches real-time gold and silver rates through API-based data extraction
- Cleans and structures the data using Pandas
- Logs historical price data to track trends and daily fluctuations over time

##  Tech Stack

- Python
- Requests (API calls)
- selenium
- webdriver manager
- Pandas (data processing)

## How to Run

### Prerequisites
- Python 3.9+
- `requests`, `pandas`, selenium and web driver managerlibraries installed

```bash
pip install requests pandas,selenium,web driver manager
```

### Run the script

```bash
python gold_silver_price_scraper.py
```

The script will fetch the latest gold and silver rates and log/update the price data automatically.
